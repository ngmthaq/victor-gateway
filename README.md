# Victor Gateway Desktop Application

## Requirement

1. Node.js [v18.16.1](https://nodejs.org/en/blog/release/v18.16.1) or later
2. Install Yarn global `npm install -g yarn`

## Scripts

Run Development:

```sh
yarn dev
```

Build For Production:

```sh
yarn build
```

Format & Lint:

```sh
yarn format && yarn lint
```
